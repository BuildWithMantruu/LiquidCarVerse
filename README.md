## ✨ Liquid Glass Car Information Platform

A premium, fully responsive web application showcasing **every car brand and model sold in India** with stunning **Apple-inspired Liquid Glass UI**, dynamic filtering, price range sliders, and real-time search functionality.

![Carverse Preview](https://via.placeholder.com/1200x630/1e4a6e/ffffff?text=Carverse+Liquid+Glass+UI)

## 🌟 Features

### 🎨 **100% Liquid Glass Design**
- Apple-style frosted glass morphism with blur effects
- Smooth animations using GSAP
- iOS-inspired interface with depth and transparency

### 🚗 **Complete Car Database**
- Every manufacturer selling in India (30+ brands)
- 57+ current models with detailed specifications
- 5+ upcoming cars with launch dates
- Realistic ex-showroom prices in ₹ INR

### 🔍 **Advanced Filtering System**
- **Brand Filtering**: Filter by any manufacturer (Maruti, Hyundai, BMW, etc.)
- **Vehicle Type**: SUV, Hatchback, Sedan, Sports, Luxury, MUV/MPV
- **Price Range Slider**: Dynamic filtering from ₹0 to ₹450+ Lakhs
- **Smart Search**: Search by car name, brand, or variant (e.g., "ZXI", "330i")

### 📱 **Premium UI/UX**
- Fully responsive design (Desktop, Tablet, Mobile)
- Interactive modal for detailed specifications
- Smooth liquid glass animations
- Real-time filtering without page reload
